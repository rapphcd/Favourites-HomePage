# Favourites HomePages

A simple Homepage to pin your favourites websites, see the meteo, date and time. Built with React and Expressjs.


# Project structure

- ``client`` : React and Vite with Tailwind.
- ``server`` : JavaScript Express Api to store favourites, background and manage meteo.

## Installation

Install this project:

```bash
git clone https://github.com/rapphcd/Favourites-HomePage
```

## Deployment

Now to deploy you need to start the front and the server.

### Client start

```bash
cd client
npm install
npm run dev
```

### Server start
```bash
cd server
npm install
npm run dev
```

And you're all done ! Enjoy.
